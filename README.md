# KnowledgeQuest

K–8 adaptive quiz app for math (physics, chemistry, and earth science planned). Questions are generated by Claude and adapt to each student's mastery level in real time.

**Stack:** Next.js 14 App Router · Supabase (PostgreSQL + Edge Functions) · Claude API · Tailwind CSS · Vercel

---

## Setup

### 1. Clone and install
```bash
npm install
```

### 2. Configure environment variables
```bash
cp .env.example .env.local
```
Fill in all values in `.env.local`:
- `NEXT_PUBLIC_SUPABASE_URL` / `NEXT_PUBLIC_SUPABASE_ANON_KEY` / `SUPABASE_SERVICE_ROLE_KEY` — from your Supabase project dashboard
- `ANTHROPIC_API_KEY` — from console.anthropic.com
- `OPENAI_API_KEY` — used for student name moderation at onboarding
- `SENDGRID_API_KEY` / `SENDGRID_FROM_EMAIL` — for weekly digest emails
- `WEEKLY_DIGEST_SECRET` / `CRON_SECRET` — random strings for endpoint protection

### 3. Run database migrations
In the Supabase dashboard → SQL Editor, run each migration in order:
```
supabase/migrations/001_initial_schema.sql
supabase/migrations/002_functions.sql
supabase/migrations/003_rls.sql
supabase/migrations/004_fix_compute_topic_score.sql
supabase/migrations/005_fix_compute_weekly_report.sql
supabase/migrations/006_fix_weekly_report_shape.sql
```

### 4. Seed topics
```bash
npm run seed        # regenerates supabase/seed.sql from topic_seed_data_v2.json
```
Then run `supabase/seed.sql` in the Supabase SQL Editor.

### 5. Start the dev server
```bash
npm run dev         # http://localhost:3000
```

## Project structure

```
app/
  (auth)/          — login + onboarding (no NavBar)
  (app)/           — quest-board + report (NavBar via group layout)
  session/[topicId]/  — quiz session (own layout, no NavBar)
  api/             — server-side API routes
lib/               — shared server logic (Claude, Supabase, guardrails, mastery, streak)
supabase/
  migrations/      — versioned SQL schema + functions + RLS
  functions/       — Deno edge function (weekly digest email)
  seed/            — topic seed data JSON
scripts/           — seed generation utility
```

See [CLAUDE.md](./CLAUDE.md) for full developer conventions, gotchas, and architecture notes.

## Deployment

Deployed on Vercel. The weekly digest cron runs every Monday at 14:00 UTC via `vercel.json`.

Set all `.env.local` variables as Environment Variables in the Vercel project dashboard before deploying.
